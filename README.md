# LawJames — Yanguéz Legal Group

Professional legal website built with React + TypeScript + Vite.

## Features
- Multilingual support (EN, ES, IT, FR, PT)
- Contact form via FormSubmit
- Service views (Corporate, Entity, Holding, Resident Agent)
- Privacy & Confidentiality Policy page
- Open Graph meta tags
- Responsive design (mobile + desktop)

## Tech Stack
- React 18 + TypeScript
- Vite
- React Router DOM
- Lucide React
- Tailwind CSS

## Dev
```bash
npm install
npm run dev
```
